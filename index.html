= The Pinball Arcade Android Users Guide

== What Is This? The Pinball Arcade Android User Guide has been created--somewhat out of frustration--to help Android users of FarSight Studio's The Pinball Arcade (TPA) app.

== Why Should I Read This? If you are a new user to the game, there are plenty of options you need to configure so your device works the way it should with TPA, but might not know how because the UI is simply not able to explain them in sufficient detail.
